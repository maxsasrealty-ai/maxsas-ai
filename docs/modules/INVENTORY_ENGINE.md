# Inventory Engine

## Real Estate Inventory Storage
- Inventory records represent projects, units, and associated metadata.
- Structured attributes support filtering by location, type, and availability.

## Project Mapping
- Inventory is mapped to project/group identifiers for portfolio-level visibility.
- Mapping enables consistent association between campaigns and inventory context.

## Campaign Usage
- Campaigns can reference inventory slices as targeting context.
- Inventory-aware workflows improve lead qualification and call relevance.
