# Agent Engine

## Purpose
- Agent Engine coordinates AI-assisted call actions and runtime decisions.

## Core Responsibilities
- Trigger call workflows with campaign and lead context.
- Capture structured responses and normalize outcome payloads.
- Forward status updates to lead, batch, and analytics modules.

## Runtime Considerations
- Supports retry-safe execution with idempotent state updates.
- Preserves traceability through linked call and runtime records.
