# Admin Engine

## Admin Dashboard
- Central control plane for tenant visibility, usage tracking, and operational metrics.
- Dashboard surfaces health, billing, and workflow status for rapid intervention.

## Client Management
- Admin tools support workspace lifecycle and user-role oversight.
- Tenant context enables scoped monitoring and controlled access operations.

## Wallet Adjustments
- Manual wallet credits/debits are applied through admin action flows.
- Each adjustment is recorded in transaction history for auditing.

## Manual Payments
- Admin can record and reconcile manual recharge events.
- Payment updates are reflected in wallet balance and ledger views.
